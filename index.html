  • Ubicomp

    Ubicomp was a CMD project in which I got the challenge to design a physical product that solves a certain problem. The problem I had to solve was about sitting to long behind a desk. I wanted my solution to be funny, so it wouldn’t annoy the user to quickly. I also wanted to make it visually attractive, so users wanted to put it on there desk. With this all in mind I designed this solution.

  • Euroute

    Euroute was a product designed in the CMD project: Project Web. The design challenge in this project was to make a web app to use while travelling. The user also needed to be able to use the web app before, during and after their journey. I came up with Euroute, a web app that helps the user to get to their location. I really liked this project because it gave me chance to combine both my technical and visual interests.
